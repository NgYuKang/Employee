##   miniproject-employee-system
This repo is a mini project coded for an interview.
## How to run
Pull the repo, then run with the following command in a terminal:<br>
./gradlew bootRun<br>
The API should be started on localhost on port 8080
## Endpoints
### Employees
- /employees GET
- /employees/{id} GET
- /employees POST
- /employees PATCH
- /employees/{id} delete
- ### Projects
- /projects GET
- /projects/{id} GET
- /projects POST
- /projects PATCH
- /projects/{id} delete
- ### departments
- /departments GET
- /departments/{id} GET
- /departments POST
- /departments PATCH
- /departments/{id} delete
